## 🎓 学习要点

### 新手必看
1. **股票代码规则**：
   - 600/601/603开头 = 沪市股票
   - 000开头 = 深市主板
   - 002开头 = 中小板
   - 300开头 = 创业板

2. **K线颜色含义**：
   - 红色（阳线）= 收盘价 > 开盘价 = 上涨
   - 绿色（阴线）= 收盘价 < 开盘价 = 下跌

3. **成交量分析**：
   - 量价齐升 = 健康上涨
   - 量价背离 = 可能反转
   - 地量地价 = 可能见底

### 进阶技巧
1. **多周期共振**：
   - 日K看趋势
   - 30分钟看节奏
   - 5分钟找买点

2. **盘口语言**：
   - 大单压盘 = 可能诱空
   - 大单托盘 = 可能护盘
   - 对倒交易 = 主力操作

3. **时间节点**：
   - 开盘15分钟 = 确定当日方向
   - 10:30 = 第一波行情结束
   - 14:30 = 尾盘冲刺时段

## 🐛 常见问题

### Q1: 点击搜索没反应？
- 检查是否输入了股票代码
- 查看浏览器控制台是否有错误
- 确认网络连接正常

### Q2: K线图显示空白？
- 等待数据加载完成（显示加载动画）
- 尝试切换其他周期
- 刷新页面重试

### Q3: 搜索出来的股票不对？
- 输入完整的6位代码
- 或者输入更精确的关键词

### Q4: 图表不能缩放？
- 确认使用的是现代浏览器
- 尝试鼠标滚轮缩放
- 或拖动底部滑块
