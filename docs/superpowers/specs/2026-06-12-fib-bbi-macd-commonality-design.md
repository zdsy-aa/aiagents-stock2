## 1. 目标与边界

独立验证两套经典买卖结构在"涨幅前期 / 下跌前期"的共性，**不与缠论买卖点挂钩**：

- **方案 A** = 斐波那契回调线 + MACD
- **方案 B** = BBI（替代 BOLL 中线）+ MACD

对每套方案**遍历其全部指标参数**，在全市场所有股票上找出："在上涨段启动前（买）/ 下跌段启动前（卖），哪些参数组合的信号有 ≥70% 的覆盖共性"。方案 A、方案 B 各自独立成榜，互不混合。

本设计**不涉及** L3 三指标挖掘的浮出（那是另一独立任务，单独立项）。

### 验证口径（已与用户敲定）
- 复用现有共性挖掘的同一把尺子：覆盖率门槛 `COVER_MIN = 0.70`。
- 事件域从"缠论买卖点"换成"ZigZag 切出来的上涨前 / 下跌前样本"。
- 每个达标参数组合**同时报提升度**，过滤"哪都亮"的废信号。

## 2. 总体流程

```
全市场日K
   │
   ├─ ZigZag 切段(pct ∈ {0.10, 0.15, 0.20}，三档各跑一遍)
   │      └─ 波谷 L / 波峰 H 拐点序列 → 上涨段 / 下跌段
   │
   ├─ 正样本池生成(W=5)
   │      ├─ 上涨前买样本 = 每个上涨段，波谷 L 前含 L 的最后 5 根
   │      ├─ 下跌前卖样本 = 每个下跌段，波峰 H 前含 H 的最后 5 根
   │      └─ 全体 bar = 提升度基线分母
   │
   ├─ 参数化信号库(遍历所有指标参数)
   │      ├─ 方案A 买 = 斐波回踩企稳(支撑) ∩ MACD金叉
   │      ├─ 方案A 卖 = 斐波反弹受阻(压力) ∩ MACD死叉   (镜像)
   │      ├─ 方案B 买 = 上穿BBI ∩ MACD金叉
   │      └─ 方案B 卖 = 跌破BBI ∩ MACD死叉              (镜像)
   │
   └─ 共性挖掘
          ├─ 逐参数组合：覆盖率 = 命中正样本段 / 正样本段总数
          ├─ 提升度 = 正样本里命中率 ÷ 全体 bar 命中率
          ├─ 筛 coverage ≥ 0.70，按提升度排序
          └─ 输出共性榜 CSV + 横向对比 md
```

## 3. 信号定义（形态固定，数值参数遍历）

形态在 brainstorming 已逐项敲定；遍历的是其中的**数值参数**。

### 3.1 ZigZag 切段
- 自最近确认拐点起，价格反向摆动幅度 ≥ `pct` 才确认新拐点。
- 波谷 L → 波峰 H = 上涨段；波峰 H → 波谷 L = 下跌段。
- `pct ∈ {0.10, 0.15, 0.20}`，三档独立各跑一遍出三套结果。
- 由构造，上涨段振幅 ≥ pct，天然只研究"涨幅 ≥ pct 的行情前期"。

### 3.2 正样本窗口（拐点后 fwd=4，"起涨/起跌初期"）
- **买样本 = 波谷 L 当根及其后 4 根 `[L, L+4]`**（含 L 共 5 根，截到该上涨段终点 H 为止）。
- **卖样本 = 波峰 H 当根及其后 4 根 `[H, H+4]`**（含 H 共 5 根，截到该下跌段终点 L 为止）。
- "命中"判据：该段对应窗口内**任意一根**信号成立，即记该段被命中。
- 覆盖率以**段**为单位：被命中段数 / 段总数。
- **口径定稿依据（2026-06-12 实测）**：原"拐点前 [L-4,L]"窗口让买信号（斐波回踩+MACD金叉本质发生在转折*之后*）几乎不亮——30股样本覆盖率≈0.09 且**提升度<1**（反相关），主榜全空。改"拐点后 [L,L+4]"后提升度升到 1.7、覆盖率 0.38。这更贴合用户原意"涨幅前期/起涨初期"（例子里 1~11 号含起涨那根 11 号）。

### 3.3 斐波回调线（方案 A）
- swing 区间：近 `N` 日最高/最低价。
- 回调位（买，支撑）：高点 − (高−低)×ratio，`ratio ∈ {0.382, 0.5, 0.618}`。
- 回踩企稳（买向）：当日最低价下探到某回调位 ±`band` 带宽内，但收盘价 ≥ 该回调位（下探不破、收回上方）。
- 镜像（卖，压力）：低点 + (高−低)×ratio 为压力位；当日最高价上探到压力位 ±band 内，但收盘价 ≤ 该压力位（反弹受阻）。
- 参数：`N ∈ {10,20,30,60}`、`ratio ∈ {0.382,0.5,0.618}`、`band ∈ {0.005,0.01,0.02}`。三条回调位各为独立参数取值（分别验证哪条最强）。

### 3.4 BBI（方案 B）
- BBI = 四条均线等权平均，周期四元组可遍历。
- 上穿 BBI（买向）：前一日收盘 ≤ BBI、当日收盘 > BBI。
- 跌破 BBI（卖向，镜像）：前一日收盘 ≥ BBI、当日收盘 < BBI。
- 参数：均线四元组 ∈ {(3,6,12,24),(5,10,20,40),(2,5,10,20)}。

### 3.5 MACD（公共项）
- 金叉（买向）：前一日 DIF ≤ DEA、当日 DIF > DEA。
- 死叉（卖向，镜像）：前一日 DIF ≥ DEA、当日 DIF < DEA。
- 参数：(fast, slow, signal) ∈ {(12,26,9),(8,17,9),(6,19,9),(5,35,5)}。

### 3.6 方案组合
- 方案 A 信号 = 斐波(N,ratio,band) **与 MACD(f,s,sig) 同日同时成立**。
- 方案 B 信号 = BBI(四元组) **与 MACD(f,s,sig) 同日同时成立**。

## 4. 参数网格规模（起点，可调）

| 方案 | 维度 | 候选 | 组合数 |
|---|---|---|---|
| A | N × ratio × band × MACD | 4×3×3×4 | 144 |
| B | BBI四元组 × MACD | 3×4 | 12 |

总跑次 = (144 + 12) × 3 档 ZigZag × (买 + 卖) = 936 次评估。每次需在全市场日K上扫描，离线跑一次。

## 7. 错误处理与边界

- ZigZag：单调或数据过短（< 最长均线周期 + N）的股票跳过并计数。
- swing 窗口：`[pivot, pivot+4]` 截到段终点 与 序列末尾 为止（末段不足 5 根取实际可得根数）。
- 提升度分母为 0（信号在全体 bar 从不触发）→ 记为 inf 但标注"无效/退化"，不入榜。
- 停牌/缺口：以实际有成交的交易日 bar 为序，不补齐自然日。
